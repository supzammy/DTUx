# Contributing to DTU Reimagined

Thank you for your interest in contributing to this project!

## How to Contribute

This is primarily a demonstration/portfolio project, but contributions are welcome for:

- Bug fixes
- Performance improvements
- Documentation updates
- Design enhancements (maintaining the Ivy League aesthetic)

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/dtu-reimagined.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`
5. Make your changes
6. Test thoroughly: `npm run dev`
7. Build to verify: `npm run build`
8. Commit with clear messages: `git commit -m "feat: add new feature"`
9. Push to your fork: `git push origin feature/your-feature-name`
10. Open a Pull Request

## Design Guidelines

Please maintain the established aesthetic:

- **Colors**: Monochrome + DTU Red (#8C1515) only
- **Typography**: Source Serif 4 (headings), Inter (body)
- **Spacing**: Professional, academic feel
- **Corners**: Sharp edges, minimal rounding

## Code Style

- Use TypeScript for type safety
- Follow existing code structure
- Keep components modular and reusable
- Add comments for complex logic
- Ensure responsive design

## Questions?

Open an issue for discussion before major changes.
